Features =========================================

  Dark mode

  Pascal syntax highlighting
  
  Easy Replace
  
  Toggleable Line Numbers
  
  Open terminal in current directory function

==================================================

Tested on 

  Linux (x86_64)

  macOS (aarch64)

  Haiku (x86_64)

  FreeBSD (x86_64)

  

Simply open the .lpi file in Lazarus and compile.
